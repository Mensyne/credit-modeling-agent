# 信贷建模Agent可视化系统
基于Streamlit+Polars构建的全流程信贷建模可视化Agent系统，覆盖**数据接入→样本划分→特征工程→模型训练→调参→评估→合规→部署→监控**全链路，支持多角色协作、多项目隔离，满足信贷监管合规要求。
## 🌟 核心功能
### 🧩 15个核心Agent模块
| 模块 | 功能说明 |
|------|----------|
| 项目管理Agent | 多用户、多角色、多项目隔离，权限控制 |
| 数据Agent | 多数据源接入：本地文件/数据库/API，Polars高性能处理 |
| 样本划分Agent | 随机划分、时间划分、OOT验证集支持 |
| 特征工程Agent | 特征衍生、筛选、分箱、编码、逐月PSI计算 |
| 模型Agent | 支持逻辑回归/XGBoost/LightGBM/CatBoost/NN，多模型对比 |
| 参数调节Agent | 可视化手动调参、贝叶斯自动调参、调参过程可视化 |
| 模型效果Agent | 分类指标、信贷专属指标（AUC/KS/PSI/Lift/Gain）、分群体评估 |
| 模型性能Agent | 训练性能、推理性能统计、多模型性能对比 |
| 可视化Agent | 特征分箱可视化、模型效果可视化、分数分箱可视化（支持多数据集/逐月） |
| 数据合规Agent | 敏感特征识别、数据源合规校验、样本合规校验 |
| 可解释性Agent | 全局/局部解释、SHAP值计算、特征贡献度展示 |
| 监管合规报告Agent | 监管报告生成、多格式导出 |
| 模型报告Agent | 全流程报告、月度监控报告、自定义模板 |
| 模型部署Agent | 模型多格式导出、一键部署生成API |
| 线上监控Agent | 效果监控、性能监控、特征漂移告警 |

## 🛠️ 快速开始
### 方式1：本地运行
1. 克隆项目到本地
2. 安装依赖：
```bash
# 安装uv（如果未安装）
pip install uv
# 安装项目依赖
uv sync
```
3. 复制配置文件：
```bash
cp .env.example .env
```
4. 启动服务：
```bash
uv run streamlit run app.py --server.port=8501
```
5. 访问系统：http://localhost:8501
默认管理员账号：`admin / admin123`
### 方式2：Docker部署
```bash
# 复制配置文件
cp .env.example .env
# 启动服务
docker-compose up -d
```

## 📖 使用流程
1. **登录系统**：使用管理员账号登录，创建普通用户账号
2. **创建项目**：按业务线/产品创建项目，分配项目成员
3. **接入数据**：支持本地文件上传、数据库对接、API接入
4. **样本划分**：选择随机划分或时间划分，拆分训练/验证/测试/OOT集
5. **特征工程**：特征筛选、衍生、分箱、编码，计算特征PSI稳定性
6. **训练模型**：选择算法，手动/自动调参，训练模型
7. **评估模型**：查看模型效果指标、性能指标，生成可视化报告
8. **部署上线**：导出模型，一键部署为API接口
9. **监控维护**：监控线上模型效果、特征漂移，定期生成监控报告
